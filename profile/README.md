# ZeroGPU

Distributed AI inference infrastructure for geo-aware edge compute. Run production inference with lower cost, lower latency, and horizontal scale across edge devices.

## Why ZeroGPU

- **Lower cost** — Inference on idle edge compute instead of GPU rentals
- **Geo-aware routing** — Requests routed to nearby capacity for lower latency
- **Edge-native models** — Nano language models (NLMs) built for edge and cloud
- **OpenAI-compatible API** — `POST /v1/responses`; integrate with familiar request shapes
- **Cloud fallback** — When edge isn’t available, the same API falls back to cloud
